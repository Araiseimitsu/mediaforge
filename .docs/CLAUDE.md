# MediaForge プロジェクト概要

## プロジェクトの目的

MediaForgeは、画像・動画・音声ファイルをブラウザ内で変換し、ローカル保存できるWebアプリケーションです。
サーバ側にファイルを送らずに完結する、社内向けのシンプルな変換体験を提供します。

## プロジェクトの目標

1. 使いやすさ: ドラッグ&ドロップで直感的に操作
2. 多様性: 画像/動画/音声の主要形式に対応
3. 安全性: 変換は端末内で完結（ローカル保存のみ）
4. UI品質: 手書き風の分かりやすい画面

## プロジェクトルール

### コーディング規約
- PythonコードはPEP 8準拠
- 1ファイルあたり1000行以内を目標
- 関数は単一責任の原則に従う
- 適切なエラーハンドリングを実装

### ファイル構成
- バックエンドはテンプレート/静的配信のみ
- フロントエンドはHTML/CSS/JavaScriptを分離
- 静的ファイルは `static/` に配置
- テンプレートは `templates/` に配置

## 活用技術

### バックエンド
- FastAPI: Webフレームワーク
- Python 3.8+: 実行環境
- Uvicorn: ASGIサーバー

### フロントエンド
- ffmpeg.wasm: ブラウザ内変換エンジン
- Tailwind CSS: スタイリング
- Vanilla JavaScript: クライアントロジック

## アーキテクチャ

### 構成
1. プレゼンテーション層: HTML/CSS/JS（templates/, static/）
2. 配信層: FastAPIによる静的配信（app/main.py）

### 変換処理
- 変換はブラウザ内で実行
- 変換後はローカル保存のみ

## セキュリティ考慮事項

1. 端末内で完結するためサーバへのファイル送信なし
2. 対応拡張子の判定はフロント側で実施
3. ブラウザ/端末の制約（メモリ/サイズ）に留意
